# 📝 My Todo App

シンプルで美しいTodoリストアプリケーション

![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

## ✨ 特徴

- 🎨 **モダンなUI** - グラスモーフィズムデザイン
- ✅ **タスク管理** - 追加・削除・完了トグル
- 🔍 **フィルター機能** - すべて/未完了/完了済み
- 💾 **自動保存** - localStorageでデータ永続化
- 📱 **レスポンシブ** - モバイル〜デスクトップ対応
- ✨ **アニメーション** - 滑らかなUI遷移

## 🚀 使い方

### インストール

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

## 🎯 機能

| 機能 | 説明 |
|------|------|
| タスク追加 | 入力欄にテキストを入力し、追加ボタンまたはEnterキーで追加 |
| タスク削除 | タスクにホバーして表示されるゴミ箱アイコンをクリック |
| 完了トグル | チェックボックスをクリックして完了/未完了を切り替え |
| フィルター | 上部のタブで表示するタスクを絞り込み |

## 🛠️ 技術スタック

- **Vite** - 高速な開発環境
- **Vanilla JavaScript** - フレームワークなしのピュアJS
- **CSS3** - グラスモーフィズム、カスタムプロパティ、アニメーション
- **localStorage API** - データ永続化
